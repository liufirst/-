 # 快捷输入HTML骨架
 输入`html：5`或者`!`  
 会自动生成以下代码，快捷方便。  
~~~~~
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>

<body>

</body>

</html>  
~~~~~

# 文件架构
文件→打开文件夹→（现在工作目录就是在所选文件夹了）   
在里面创建文件夹（例如css）就很方便

# VScode必备插件
|插件                 |         作用                   |
|:-------------------:|-------------------:|
|chinese|中文汉化包|
|Open  in browser|右键打开html|
|JS-CSS——HTML formatter|ctrl+s格式化代码|
|Auto Rename tag|自动重命名配对的html标签|
|css peek|追踪到样式|
